# 启用 GitHub Pages 文档网站

## 📝 步骤

### 1. 进入仓库设置
1. 访问 https://github.com/rrairr0117-a11y/walitts
2. 点击 "Settings"

### 2. 配置 Pages
1. 左侧菜单找到 "Pages"
2. Source 选择 "Deploy from a branch"
3. Branch 选择 "main"
4. Folder 选择 "/docs"
5. 点击 "Save"

### 3. 等待部署
- 等待 1-2 分钟
- 页面会显示访问地址

### 4. 访问文档网站
您的文档网站地址将是：
```
https://rrairr0117-a11y.github.io/walitts/
```

## ✨ 文档网站功能

- 📖 项目介绍和特性展示
- 💻 下载链接（自动链接到 Releases）
- 🛠️ 技术栈展示
- 📚 快速开始指南
- 🏗️ 架构说明
- 🤝 贡献指南

## 🎨 自定义

如需修改文档内容，编辑 `docs/index.html` 文件，然后推送到 GitHub 即可自动更新。
